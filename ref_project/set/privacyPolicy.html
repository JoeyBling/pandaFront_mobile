张网分销隐私权政策

概述和声明

张网分销（以下简称“我们”）尊重并保护用户隐私，我们将按照本隐私权政策（以下简称“本政策”）收集、使用及共享您的个人信息。本政策包含了我们收集、存储、保护、使用和共享 您的个人信息的条款，我们建议您完整的阅读本政策，以帮助您了解维护自己隐私权的方式。为了使您充分理解本政策的内容，本政策中与您的权益存在或可能存在重大关系的条款。当您使用我们提供的任意服务时，即表示您已同意我们按照本政策来合法使用和保护的个人信息。

我们收集、使用您的信息是出于遵守国家法律法规的规定以及向您提供服务及提升服务质量的目的（包含支持我们开发新产品或完善已有产品功能，为您和其他用户提供更为优质的服务）。如您有任何问题可以和我们取得联系。
